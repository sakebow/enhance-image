# 图像增强

主要是将多个图片组合成一个大图片，适用于图片较小的情况，比如数字水印中的文字图片、一些自制的256x256的图片等。

# 功能

- [x] 对图片进行旋转
  - [x] 90度
  - [x] 180度
  - [x] 270度
- [x] 对图像进行翻转
  - [x] 水平翻转
  - [x] 垂直翻转
- [x] 对图像进行组合
  - [x] 随机抽取图片
  - [x] 组合为$n{\times}n$的正方形图片
  - [x] 可以指定$n$的大小
- [x] 对组合的图像进行增强
  - [x] 增加高斯噪声
  - [x] 修改图片亮度
- [x] 参数可配置
  - [x] 使用`yaml`文件参数配置参数
  - [x] 通过创建对象的时候传值进行初始化
